# RASP_ZERO-
Rasp(Runtime Application Self Protection) 配套的简单靶场。

启动方式：
1.idea配置好tomcat

2.VM Options 设置为：
-javaagent:C:\Users\****\RASP_0.jar  （此处填写javaagent jar包路径）
